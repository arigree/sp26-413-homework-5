### CSV & PapaParse

A CSV file is a text format used to store tabular data where each line represents a row and each value is seperated by a common. It's used because it is lightweight and easy to read. PapaParse is a JavaScript library that reads CSV files and converts them into JavaScript objects. PapaPrase handles parsing, headers, and formatting automatically.

### KPI Calculation

The .reduce() medthod in JavaScript is used to iterate over an array and get a single result from all the elements. It takes a callback function and an initial value then applies that function to each item in the array while also counting however many times it does so. Its useful for summing values, counting items, or getting averages from a dataset. .reduce() lets you process all the rows of data in a single pass through the array.
